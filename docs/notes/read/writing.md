# Writing

我意识到，阻挡自己的是个人财富。

因为没有财富，才给不了喜欢的人足够的安全保证，自己选择终身不婚，其实是一种逃避，在争取财富上选择懦弱。

我不得不承认，人必须要现实点，理想主义只是空想臆测。我觉得自己不能再像一个男孩了，应该敢于直面现实的经济问题，以一个男人的姿态，无所畏惧，顶起责任。

男孩可以逃避，可以淘气，可以软弱，可以玩忽所有。男人，却不行。
男孩只存在于象牙塔或者幻想中，在现实中，唯有男人。

错过的女生，可以给你心伤，也给你成长。
